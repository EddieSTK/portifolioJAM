Quente ou frio?

O projeto é um jogo criado com Javascript, focado em achar um ponto que muda toda vez que o jogo é reiniciado.

O jogo em si é levemente diferente da versão normal do Alura, mas a grande parte é a mesma.

A parte do código se resume em randomizar a localização de um ponto, fazer uma área em volta do mouse crescer e diminuir, e, quando encosta no ponto, o jogo para.

É um dos primeiros projetos que fiz durante o 2° Ano do EM.

Corrida Maluca!

O projeto é um mini-jogo de corrida em que até quatro pessoas podem participar, cada uma usando uma seta do teclado.

O jogo é quase o mesmo se comparado com a versão da Alura, mas usa alguns elementos diferentes, como os ícones dos jogadores.

O código consiste no movimento dos jogadores, uma checagem para verificar se o jogador já clicou na tela, um sensor de quem chegou primeiro, e a tela do vencedor.

Mesmo que seja simples, foi legal tirar um tempo para jogar isso com os colegas.
